# 逻辑说明

## 1. 防御信号

每个指标根据配置表判断是否触发防御信号：

- `moving_avg + below`：最新值低于过去 N 期移动平均；
- `moving_avg + above`：最新值高于过去 N 期移动平均；
- `mom + down`：最新值低于上期；
- `mom + up`：最新值高于上期。

## 2. 加权防御分数

一级维度得分 = 该维度触发指标数 / 该维度有效指标数。

总防御分数 = 各一级维度得分 × 维度权重 后求和。

## 3. 回测

回测按月计算历史防御分数，并统计不同分数区间下各资产未来 1/3/6 个月收益表现。动态仓位策略使用上月防御分数决定本月仓位，以避免未来函数。

## 4. AI 解释

DeepSeek API 只接收当前信号摘要和回测摘要，用于生成文字解释，不参与信号计算。
